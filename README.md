Introducing "Nize",
your go-to solution for adding a touch of elegance to your Flutter projects!

# Getting started

In your flutter project add the dependency:

```yaml
dependencies:
  nize: ^0.0.1
```

Import the package:

```dart
import 'package:nize/nize.dart';
```

# Usage

### NizeColors

```dart
Widget myContainer() {
 return Container(
    color: NizeColors.purple.lavenderWhip,
  );
}
```
